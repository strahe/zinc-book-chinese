# 开始

## 安装

根据一下步骤，使用 Zinc 框架:

1. 下载适用于您的操作系统和架构的最新版本.
2. 将包含二进制文件的文件夹添加到 `PATH`.
3. 通过您喜欢的终端使用二进制文件.

最小化的 Zinc 框架由三个工具组成：

- `zargo` 包管理器
- `znc` Zinc 编译器
- `zvm` Zinc 虚拟机

`zargo` 可以通过作为子进程使用编译器和虚拟机，所以你只需要 `zargo` 来处理你的项目。

有关 `zargo` 的更多信息， 打开这个 [章节](../09-zargo/00-overview.md).

## Visual Studio 编码扩展工具

Zinc 有一个语法高亮扩展程序叫作： “Zinc Syntax Highlighting”.
当你打开 Zinc 文件后，IDE 应该建议您安装它！

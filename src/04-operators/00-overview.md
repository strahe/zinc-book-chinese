# 优先级

首先执行最上面的。

|              运算符              |    关联性    |
|------------------------------------|---------------------|
| ::                                 | 从左到右       |
| [] .                               | 从左到右       |
| - ~ !                              | 一元               |
| as                                 | 从左到右       |
| * / %                              | 从左到右       |
| + -                                | 从左到右       |
| << >>                              | 从左到右       |
| &                                  | 从左到右       |
| ^                                  | 从左到右       |
| ⎮                                 | 从左到右       |
| == != <= >= < >                    | 需要括号 |
| &&                                 | 从左到右       |
| ^^                                 | 从左到右       |
| ⎮⎮                               | 从左到右       |
| .. ..=                             | 需要括号 |
| = += -= *= /= %= ⎮= ^= &= <<= >>= | 需要括号 |
